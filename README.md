# 🗒️ Sistema de Cadastro em Python

Projeto simples de cadastro de usuários feito em Python.

---

## 🚀 Funcionalidades

- Cadastrar usuário
- Listar usuário
- Excluir usuário
- Implementar o JSON

## 🛠️ Tecnologias utilizadas

* Python
* json

## 🖳 Como executar

1. Abra o terminal na pasta do projeto

2. Execute:

python main.py
